# 📌 Lifebase  
A simple and elegant **To-Do List Application** built with **Node.js, Express, and PostgreSQL**.  
Lifebase helps you organize your daily tasks with options to add, edit, and delete items.  

---

## 🚀 Features
- ✅ Add new tasks  
- ✏️ Edit existing tasks  
- ❌ Delete completed tasks  
- 💾 Data persistence with PostgreSQL  
- 🎨 Clean, responsive UI  

---

## 🛠️ Tech Stack
- **Frontend**: EJS(Template Engine), CSS, Vanilla JS  
- **Backend**: Node.js, Express.js  
- **Database**: PostgreSQL
